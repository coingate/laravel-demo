# CoinGate PHP Example with Laravel

## Requirements
- PHP 8.1+
- CoinGate API Key

## Installation

- clone this repository
- run `cp .env.example .env`
- run `composer install`
- run `php artisan serve`

- [CoinGate API Documentation](https://developer.coingate.com/)
